Safety Boundary

Public repo hygiene

This repo uses approved public fixtures only. It does not include client drawings, private opportunity notes, credentials, tokens, proprietary files, or license-uncertain assets.

The native examples are handoff scaffolds for the licensed CAD environment, not a claim that every runtime behavior can be validated outside the product.

This CAD library is in development. This is an early public preview for feedback on the best business case, workflow shape, and proof path.
