# Runbook

1. Run `npm run doctor`.
2. Run `npm run verify`.
3. Run `npm run demo`.
4. Run `dotnet build quickstart`.
5. Run `dotnet run --project quickstart`.
6. Review `reports/demo-validation-report.json` and `reports/quickstart-report.json`.
7. Move to native/runtime proof only inside an approved private project with licensed tooling.
